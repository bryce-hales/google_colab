# SQL Coursework

This folder contains SQL assignments, labs, templates, and query result exports from the Data Technology program.

## Focus Areas

- Basic SQL syntax
- SELECT queries
- Filtering with WHERE
- Sorting and limiting records
- Aggregate functions
- GROUP BY and HAVING
- Joins
- Data manipulation
- SQL functions
- Query templates and completed assignment scripts
- Working with sample relational databases in DBeaver or SQLite

## Files

| File | Topic |
| --- | --- |
| `assignments/assignment-03-queries.sql` | Basic SQL query practice. |
| `assignments/assignment-04-terminology-and-syntax.sql` | SQL terminology, syntax, and simple SELECT/WHERE practice. |
| `assignments/assignment-05-filters.sql` | Filtering practice using WHERE, IS, and LIMIT. |
| `assignments/assignment-06-aggregates-template.sql` | Aggregate query practice using COUNT, date ranges, and grouping. |
| `labs/lab-02-filters-demo.sql` | Filter demo lab. |
| `labs/lab-02-filters-demo-template.sql` | Filter demo lab template. |
| `exports/customer-contact-info.csv` | Query result export from customer contact data. |
| `exports/sales-sales-person-sample.csv` | Query result export from sales person data. |

## Local Database Resources

Some coursework used local SQLite database files, including sample sales, purchasing, product, restaurant, music, weather, and bowling league databases. These files are excluded from version control when appropriate because database files can be large and are often used as local course resources rather than final portfolio artifacts.
