# SQL Coursework

This folder contains SQL assignments, labs, templates, and small query result exports from the Data Technology program.

## Focus Areas

- Basic SQL syntax
- SELECT queries
- Filtering with WHERE
- Sorting and limiting records
- Aggregate functions
- GROUP BY and HAVING
- Query templates and completed assignment scripts
- Working with sample relational databases in DBeaver or SQLite

## Files Added

| File | Purpose |
| --- | --- |
| `assignments/assignment-03-queries.sql` | Basic SQL query practice. |
| `assignments/assignment-04-terminology-and-syntax.sql` | SQL terminology, syntax, and simple SELECT/WHERE practice. |
| `assignments/assignment-05-filters.sql` | Filtering practice using WHERE, IS, and LIMIT. |
| `assignments/assignment-06-aggregates-template.sql` | Aggregate query template for COUNT, date ranges, and grouping. |
| `labs/lab-02-filters-demo.sql` | Completed filter demo lab. |
| `labs/lab-02-filters-demo-template.sql` | Blank/template version of the filter demo lab. |
| `exports/customer-contact-info.csv` | Small query result export from customer contact data. |
| `exports/sales-sales-person-sample.csv` | Small query result export from sales person data. |
