# Statistics and Spreadsheet Coursework

This folder documents spreadsheet-based assignments and practice files from the Data Technology program.

## Focus Areas

- Descriptive statistics
- Histograms and frequency distributions
- One-sample z tests
- T-tests
- ANOVA
- Hypothesis testing
- Scatter plots
- Pivot tables
- Basic data cleaning and spreadsheet formulas
