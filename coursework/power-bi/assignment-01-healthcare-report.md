This assignment focused on building a Power BI report using healthcare-style data. The report files and datasets include patient records, lab work records, and related report files.

## Skills Practiced

- Importing healthcare-style CSV data into Power BI
- Building visuals from patient and lab-related fields
- Working with categorical and numerical fields
- Summarizing measures such as billing amounts, coverage, BMI, length of stay, and severity
- Creating report pages from structured datasets
