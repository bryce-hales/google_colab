# CloudNine Flight Performance Dashboard

This project focused on building a branded Power BI dashboard for flight delay performance across the United States.

## Dashboard Summary

The CloudNine dashboard includes visuals for:

- Late status of flights
- Relationship between arrival delay and departure delay
- Percentage of late flights by state
- Count of flights by month and late status
- Average arrival delay and number of flights by airline
- Airline and flight status slicers

## Insights Documented

The written report explains that departure delays often lead to arrival delays and that delay rates vary by airline, month, and state. The dashboard uses interactive filters and drill-down features so users can explore airline performance, seasonal trends, and geographic delay patterns.
