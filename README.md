# Data Technology Coursework Portfolio

This repository organizes coursework and practice files from a Data Technology program.

## Purpose

The goal of this repository is to document coursework across programming, analytics, statistics, databases, visualization, and project-based learning.

## Repository Structure

| Folder | Focus |
| --- | --- |
| `coursework/python/` | Python notebooks, notes, labs, and project work. |
| `coursework/machine-learning/` | Model building, evaluation, and related notebooks. |
| `coursework/r/` | R scripts, ggplot2 practice, and R Markdown reports. |
| `coursework/sql/` | SQL scripts, labs, query exports, and database notes. |
| `coursework/statistics-spreadsheets/` | Spreadsheet labs, workbook documentation, and statistics practice. |
| `coursework/tableau/` | Tableau workbook documentation and dashboard practice. |
| `coursework/power-bi/` | Power BI report documentation, screenshots, filters, charts, and dashboard practice. |
| `coursework/datasets/` | Shared datasets used across coursework. |

## Topics Covered

- Python fundamentals
- Data cleaning and exploratory analysis
- R programming and visualization
- SQL querying and database concepts
- Spreadsheet analysis and statistics
- Machine learning fundamentals
- Power BI reports and dashboards
- Tableau dashboards and visual analytics

## Skills Practiced

- Google Colab notebook workflows
- Python and R programming
- pandas, NumPy, tidyverse, and ggplot2
- SQL querying
- Spreadsheet formulas, charts, and pivot tables
- Descriptive and inferential statistics
- Data visualization and dashboard design
- Power BI and Tableau reporting
